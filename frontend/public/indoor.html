How It Works

Multi-Floor QR Code System

For multi-story buildings, QR codes include floor information in this format:

location_name|x|y|floor

Example: conference|30|15|1 for a conference room on the first floor

Reception (Floor 0)

Conference (Floor 1)

Manager (Floor 2)

Navigation Process

  • Scan a QR code to set your current position and floor
  • Select your destination from the dropdown menu
  • Start navigation to see AR directions
  • If destination is on another floor:
    • System guides you to the nearest stairs/elevator
    • After changing floors, scan a QR code on the new floor
    • Navigation continues to your destination
  • Follow the green arrow in your camera view

Implementation Notes

  • Works on mobile devices with cameras
  • No installation required - runs in any modern browser
  • QR codes placed at key locations on each floor
  • Coordinates are relative to each floor's layout
  • Staircases/elevators are predefined transition points
  • Arrow points toward destination using device orientation
